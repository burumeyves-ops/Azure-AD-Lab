# Phase 1: Azure Foundation

[← Back to main README](../README.md)

## Objective
Set up the base Azure environment for my IAM lab and create a Windows Server VM that I can manage remotely.

## What I Built
- Azure resource group
- Virtual network and subnet
- Windows Server VM
- Remote Desktop connection to the VM

## Why This Matters
This phase is the foundation for the rest of the lab. I need a working Azure VM before I can install Active Directory Domain Services, test identity workflows, or build PowerShell automation.

## Step-by-Step Notes
1. Created a resource group
2. Created a virtual network and subnet
3. Created a Windows Server VM
4. Enabled RDP access
5. Connected to the VM
6. Verified login
